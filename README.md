# NPK-SRBMIRM-Sample-Multispectral-Data
Sample multispectral UAV images associated with UAV-Based Spectral Residual Attention Network for Non-Destructive Estimation of Nitrogen, Phosphorus, and Potassium in Direct-Seeded Rice (DSR) and Transplanted Rice (TPR).
The complete dataset is available from the corresponding author upon reasonable request.
